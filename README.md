# Image Caption Generator - React Frontend

AI-powered Image Caption Generator using Deep Learning (InceptionV3 + LSTM).

## Quick Start
1. Extract the ZIP file
2. cd image-caption-generator-frontend
3. npm install
4. npm start
5. Open http://localhost:3000

## Connect Backend
Edit src/App.js and add your API endpoint at line 20.

## Features
- Beautiful gradient UI
- Drag-and-drop upload
- Fully responsive
- Real-time processing

Built for Deep Learning Major Project.
